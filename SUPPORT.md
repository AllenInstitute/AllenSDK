# Support

Thanks for using AllenSDK.

AllenSDK is in **selective maintenance mode**.
Maintainers prioritize bug fixes, security fixes, documentation/tests, and targeted compatibility updates.

## Where to ask for help

### 1) General questions and discussion
For general Allen Brain Map / data-tool questions, usage discussion, and community help, please use:

- **Allen Brain Map Community Forum**: https://community.brain-map.org

### 2) Bug reports
If you found a reproducible bug in AllenSDK, open a GitHub issue:

- **GitHub Issues**: https://github.com/AllenInstitute/AllenSDK/issues

Please include:
- minimal reproducible example
- expected vs actual behavior
- full traceback/error output
- environment details (OS, Python, AllenSDK version, install method)

### 3) Feature requests
Feature requests are welcome, but please note:

- Maintainers are not planning new feature development.
- Community feature PRs are welcome.
- Requests may be closed as **not planned** if maintenance burden is too high.

Open feature requests here:

- **GitHub Issues**: https://github.com/AllenInstitute/AllenSDK/issues

## Before opening an issue

Please check:

- [Existing issues](https://github.com/AllenInstitute/AllenSDK/issues)
- [Documentation](https://allensdk.readthedocs.io/en/latest/)
- [Contribution guide](CONTRIBUTING.md)

## Response expectations

Maintainer response times vary by capacity.
Issues and PRs may be triaged as `accepted`, `needs-info`, `help wanted`, or `not planned`.

If your issue has all requested details and has not received a response after a reasonable period, a polite follow-up comment is welcome.

## Contributing fixes

If you want to contribute a fix or improvement, please see [CONTRIBUTING.md](CONTRIBUTING.md).
